# 💜 Mentoria QA — Heart Delas

Bem-vindas ao repositório oficial da **Mentoria de QA** do **He4rt Delas**! 🚀

Este espaço foi criado com muito carinho para reunir **materiais de estudo**, **desafios práticos** e **recursos de apoio** para todas as participantes que estão iniciando ou evoluindo suas carreiras em **Quality Assurance**.

---

## 🎯 Sobre a Mentoria

A Mentoria QA do Heart Delas é um programa voltado para **mulheres que querem ingressar ou crescer na área de testes de software**. Aqui você vai aprender desde os fundamentos de QA até técnicas mais avançadas, sempre com apoio, troca de experiências e muita prática.

### O que você vai encontrar neste repositório:

- 📚 Materiais teóricos sobre QA
- 🧪 Desafios práticos (manuais e automatizados)
- 🐛 Exercícios de bug report e reprodução
- 🛠️ Indicações de ferramentas essenciais
- 📖 Links de estudo complementares
- 💡 Dicas de carreira e mercado

## 🧩 Como participar dos desafios

1. **Acesse a pasta `desafios/`** e escolha o desafio da semana.
2. **Leia atentamente** o README de cada desafio.
3. **Desenvolva sua solução** (bug report, caso de teste, script, etc.).
4. **Envie sua entrega** podendo ser criando um repositorio no git, Jira, o que for melhor para ti. 
5. **Participe das discussões** com as colegas e compartilhe aprendizados!

## 📌 Regras de convivência

Para que a mentoria seja um ambiente **seguro, acolhedor e produtivo**, pedimos que todas sigam:

- 💜 **Respeito e empatia** acima de tudo.
- 🙋‍♀️ **Não existe pergunta boba** — estamos aqui para aprender juntas.
- 🤝 **Colaboração** é mais importante que competição.
- ⏰ **Comprometimento** com prazos e entregas.
- 🔒 **Confidencialidade**: o que é discutido no grupo, fica no grupo.
